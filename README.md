# Urinary Sediment Dataset
a dataset consisting of 5,376 annotated images corresponding to 7 categories of urinary particle

- cast
- cryst(crystals)
- epith(epithelial)
- epithn(epithelial nuclei)
- eryth(erythrocyte) 
- leuko(leukocyte)
- mycete

```sh
/VOCdevkit
└── Urinary Sediment Dataset
    ├── Annotations
    ├── ImageSets
    │   └── Main
    │       ├── test.txt
    │       ├── train.txt
    │       └── val.txt
    └── JPEGImages
```


- train set: 4256
- val set: 852
- test set: 268
